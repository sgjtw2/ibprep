3-Statement Model
IS Net Income feeds into BS Retained Earnings
D&A and Capex link IS, BS, and CFS
Balance sheet must balance at all times
CFS ending cash must equal BS cash
DCF Valuation
UFCF = NOPAT + D&A − Capex − ΔNWC
WACC = CoE × E% + CoD(1−t) × D%
Terminal value typically 60–80% of EV
Always run sensitivity on WACC and terminal assumptions
EV − Net Debt → Equity Value → Price per Share
DCF Key Ratios
FCF Yield = UFCF / EV
FCF Conversion = UFCF / Net Income (target >90%)
TV / EV ratio (>80% signals high uncertainty)
Implied EV/EBITDA as sanity check on price
LBO Model
Entry EV = EBITDA × Entry Multiple
Sources = Uses (S&U must balance)
IRR = MOIC^(1/hold years) − 1
Target: 20–25%+ IRR, 2.5–3.5x+ MOIC
De-lever to <4.0x Debt/EBITDA by exit
Coverage ratio >2.0x throughout hold
M&A / Accretion-Dilution
Accretive if PF EPS > Acquirer standalone EPS
Key drivers: synergies, shares issued, D&A step-up
All-stock deals dilute EPS via new share issuance
PPA creates non-cash D&A drag on PF earnings
Synergy NPV = key value creation metric
Breakeven synergies = minimum credible threshold
Restructuring
Fulcrum = security that transitions from full → partial recovery
Absolute Priority Rule: senior before junior, no exceptions
Fulcrum holders control the plan of reorganisation
Distressed debt trades at recovery expectation, not face
Going concern EV > liquidation value — preserve optionality
Post-reorg: right-size debt so leverage < 4–5x EBITDA
Comps Analysis
Use LTM financials for all comps — consistency is key
EV/EBITDA is primary multiple; EV/Rev for high-growth
Transaction comps carry 20–40% control premium
Use median, not mean — outliers skew the mean
Rule of 40 = Revenue Growth % + EBITDA Margin %
Cross-check comps implied price vs DCF range
